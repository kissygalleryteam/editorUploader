## editorUploader
kissy editor的图片上传由于是使用flash上传，很有可能会出现session丢失的bug，所以写了这个插件用于替换editor的上传插件。

v1.4基于kissy1.4，使用[uploader1.5](http://gallery.kissyui.com/uploader/1.5/guide/index.html)。


* 版本：2.0.0
* 作者：明河
